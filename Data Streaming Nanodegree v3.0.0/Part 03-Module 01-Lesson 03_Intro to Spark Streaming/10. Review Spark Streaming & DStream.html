How Can We Form DStreams?

How can we form DStreams? (may be more than one answer)
SOLUTION:
  • DStreams can be created from live data (such as data from Kafka or Flume)
  • DStreams can be created by parallelizing data from local files
  • Dstreams can be generated by transformation of existing DStreams, using operations such as map, window and reduceByKeyAndWindow.
