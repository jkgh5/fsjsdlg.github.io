What is the correct way to use a StructType called vehicleStatusSchema to deserialize JSON?
SOLUTION: ``` vehicleStatusStreamingDF \ .withColumn("value",from_json("value",vehicleStatusSchema)) ```

Select an appropriate Instantiation of a StructType for the following VehicleStatus JSON: 

{
  "truckNumber":"5169",
  "destination":"Florida",
  "milesFromShop":505,
  "odomoterReading":50513
}
SOLUTION:
  • ``` vehicleStatusSchema = StructType ( [ StructField("truckNumber", StringType()), StructField("destination", StringType()), StructField("milesFromShop", IntegerType()), StructField("odometerReading", IntegerType()) ] ) ```

Which StructField instantiations below match the VehicleCheckin JSON:

{
  "reservationId":"1601485848310",
  "locationName":"New Mexico",
  "truckNumber":"3944",
  "status":"In"}
SOLUTION:
  • ` StructField("locationName", StringType())`
  • `StructField("status", StringType()`
