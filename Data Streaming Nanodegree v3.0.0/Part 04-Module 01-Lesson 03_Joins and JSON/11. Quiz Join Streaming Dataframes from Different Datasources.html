Identify the Common Field

Identify the common field in the following payloads best suited to use for a join expression:

Reservation

{
  "reservationId":"814840107",
  "customerName":"Jim Harris", 
  "truckNumber":"15867", 
  "reservationDate":"Sep 29, 2020, 10:06:23 AM"
}

Payment

{
  "reservationId":"9856743232",
  "customerName":"Frank Aristotle",
  "amount":"946.88"
}
SOLUTION: `reservationId`
