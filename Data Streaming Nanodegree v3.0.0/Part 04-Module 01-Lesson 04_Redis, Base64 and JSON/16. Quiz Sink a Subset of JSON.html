Quiz: Sink a Subset of JSON Fields with Pyspark

QUIZ QUESTION::

Match the Spark SQL Function with its purpose

ANSWER CHOICES:



to_json

from_json

unbase64

base64

split
SOLUTION:

split

from_json

base64

to_json

unbase64

If I wanted to extract the number portion of the street address in the data below, what Spark SQL function would I use?

{"businessName":"Sven's Bread",
"streetAddress": "9876 S Fenton Ave.",
"city":"New York",
"state":"New York"
}
SOLUTION: `split`

I have a DataFrame called testCustomersDF containing the fields: firstName,lastName, streetAddress,phoneNumber,customerId. I want to sink the fields customerId and phoneNumber to a Kafka Topic called test-customers on a Broker listening at broker1.testbroker.net:9092, what is the syntax?
SOLUTION: ``` testCustomersDF \ .selectExpr( "cast(customerId as string) key", "cast(phoneNumber as string) value" ) \ .writeStream \ .format("kafka") \ .option("kafka.bootstrap.servers", "broker1.testbroker.net:9092")\ .option("topic", "test-customers")\ .option("checkpointLocation","/tmp/kafkacheckpoint")\ .start()\ .awaitTermination() ```
