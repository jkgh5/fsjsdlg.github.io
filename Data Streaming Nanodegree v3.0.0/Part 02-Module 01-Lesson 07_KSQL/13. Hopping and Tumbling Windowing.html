Hopping and Tumbling Windowing

Hopping and Tumbling Windowing

  • KSQL supports Tumbling windows with the WINDOW TUMBLING (SIZE <duration>) syntax
  • KSQL supports Hopping windows with the WINDOW HOPPING (SIZE <duration>, ADVANCE BY <interval>) syntax

Which parameters are required to specify a Hopping Window?

Which parameters are required to specify a Hopping Window in KSQL? (more than one answer)
SOLUTION:
  • `SIZE `
  • `ADVANCE BY `

Which best describes a tumbling window?

How might you approximate a Sliding Window in KSQL?
SOLUTION: Use a Hopping Window with a short `ADVANCE` period
