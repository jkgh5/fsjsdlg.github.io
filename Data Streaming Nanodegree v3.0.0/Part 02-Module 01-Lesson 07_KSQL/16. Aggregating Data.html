ND0029 C01 L07 16 Aggregating Data

Aggregating Data

KSQL Aggregations

  • Use GROUP BY to create aggregations in KSQL
  • GROUP BY always creates a KSQL Table
  • KSQL supports aggregations like COUNT, MAX, MIN, SUM, TOPK, HISTOGRAM and more

What types of aggregations are available in KSQL?

What types of aggregations are available in KSQL?
SOLUTION:
  • COUNT
  • HISTOGRAM
  • MAX
  • TOPK
  • MIN

The result of a group by is always a table

The result of a group-by is always a table.
SOLUTION: True
