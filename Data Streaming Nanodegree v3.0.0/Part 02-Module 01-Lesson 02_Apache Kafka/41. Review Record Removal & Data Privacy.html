41. Review: Record Removal & Data Privacy

Which scenarios would be appropriate for using message expiration to manage record removal?

Which scenario(s) would be appropriate for using message expiration to manage record removal, if our goal is to delete records within 24 hours? (may be more than one answer)
SOLUTION:
  • A purchases event topic with a 12 hour expiration
  • A user click event topic with a 20 hour expiration

Which scenarios would be appropriate for using log compaction?

Which scenarios would be appropriate for using log compaction to remove user data? (may be more than one answer)
SOLUTION:
  • Text message events are stored in a topic keyed on user id
  • Purchase data is stored in a topic keyed on user ID

Which of the following best describes the user key encryption scenario?

Which of the following best describes the User Key Encryption strategy?
SOLUTION: Create a topic with an encryption key for every user, encrypt all other topic data for the user with that key
