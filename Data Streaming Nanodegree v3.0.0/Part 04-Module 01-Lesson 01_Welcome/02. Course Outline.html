Course Outline Summary

Lesson 1 Overview

In your first lesson, you will begin with Streaming DataFrames in Spark, including:

  • Starting a Spark Cluster
  • Deploying a Spark Application
  • Creating a Streaming DataFrame with Kafka
  • Creating and Querying a Spark View

Lesson 2 Overview

In the second lesson, you will work with Joins and JSON, including:

  • Parsing a JSON Payload into separate fields for analysis
  • Joining two streaming DataFrames
  • Writing a streaming dataframe to Kafka

Lesson 3 Overview

In the third and last lesson, you will work with Redis, Base64, and JSON, including:

  • Manually saving to Redis
  • Reading Redis Data from a Kafka Topic
  • Parsing Base64 encoded information
  • Sinking a subset of JSON fields.

Project Overview

In the Project you complete at the end of the course, you will evaluate human balance, applying techniques from the course to a real-world streaming analytics application.
