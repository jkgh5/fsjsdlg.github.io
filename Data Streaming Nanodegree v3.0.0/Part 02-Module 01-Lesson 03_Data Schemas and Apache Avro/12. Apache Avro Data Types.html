12. Apache Avro Data Types

ND0029 C01 L03 11 Apache Avro Data Types

Avro Types

Avro Types

  • Full documentation is available on the Avro website
  • Primitive Types should be familiar, as they closely mirror the built-in types for many programming languages.
    • null
    • boolean
    • int
    • long
    • float
    • double
    • bytes
    • string
  • Complex Types allow nesting and advanced functionality.
    • records
    • enums
    • maps
    • arrays
    • unions
    • fixed

Which of the following are supported primitive types?

Which of the following are supported primitive types in Avro? (more than one answer)
SOLUTION:
  • boolean
  • double
  • bytes
  • float
  • int
  • null

Which of the following are supported complex types?

Which of the following are supported complex types in Avro? (more than one answer)
SOLUTION:
  • `enum`
  • `union`
  • `record`
  • `map`
  • `array`
  • `fixed`

Which of the following is a properly defined Avro record?

Which of the following is a properly defined Avro record?
SOLUTION: ``` { "type": "record", "name": "email_sent", "namespace": "com.udacity.lesson3", "fields": [ {"name": "email", "type": "string"}, {"name": "sent_at", "type": "int"}, {"name": "body", "type": ["string", "null"]} ] } ```
