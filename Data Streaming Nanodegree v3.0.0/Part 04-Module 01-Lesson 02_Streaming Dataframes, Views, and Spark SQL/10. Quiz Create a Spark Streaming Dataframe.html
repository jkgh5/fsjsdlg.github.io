SOLUTION:

  • A Kafka Topic can be compared to a mailbox

Which of these is a correct example of reading from a Kafka source?
SOLUTION: ``` kafkaRawStreamingDF = spark \ .readStream \ .format("kafka") \ .option("kafka.bootstrap.servers", "localhost:9092") \ .option("subscribe","balance-updates") \ .option("startingOffsets","earliest")\ .load() ```
