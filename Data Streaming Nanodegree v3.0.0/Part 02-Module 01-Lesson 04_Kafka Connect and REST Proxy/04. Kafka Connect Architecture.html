Kafka Connect Architecture

ND0029 C01 L04 04 The Connect Framework

Which of the following best describes the Kafka Connect architecture?

Which of the following best describes the Kafka Connect architecture?
SOLUTION: One or more Java web servers using plugin JARs to bridge Kafka and other external data systems

Which of the following does Kafka Connect support?

Which of the following actions does Kafka Connect support? (may be more than one answer)
SOLUTION:
  • Sourcing data from external data sources into Kafka
  • Sinking data out of Kafka into external destination data sources

Why are Connectors not built into the Web Server?

Why are Connectors not built into the web server directly?
SOLUTION: The plugin architecture allows developers to create functionality that the original designers may not have imagined
