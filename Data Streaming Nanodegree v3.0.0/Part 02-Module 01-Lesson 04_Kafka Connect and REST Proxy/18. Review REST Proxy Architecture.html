18. Review: REST Proxy Architecture

REST Proxy can create topics (true/false)

REST Proxy can create topics
SOLUTION: False

Which of the following types of interactions does REST Proxy support?

Which of the following types of interactions does REST Proxy support? (may be more than one answer)
SOLUTION:
  • Fetch administrative and metadata information
  • Consumer from topics
  • Produce data to an existing topic

In what scenarios would you not want to use REST proxy?

In what scenarios would you not want to use REST Proxy? (may be more than one answer)
SOLUTION:
  • You want to put logs into Kafka from a server without a Kafka integration
  • When writing a Python application that can be updated to use new dependencies
  • When writing a Java application that can be updated to use new dependencies

In what scenarios would you want to use REST Proxy?

In what scenarios would you want to use REST proxy? (may be more than one answer)
SOLUTION:
  • When working on a frontend client without a native Kafka library
  • When working on a legacy application that supports HTTP but can’t add new dependencies
  • For reading administrative and metadata information about the cluster
